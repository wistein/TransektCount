Installation of TransektCount and TourCount


1. Install the apps from F-Droid store (released versions, recommended)
-----------------------------------------------------------------------

   Get the apps without docs and species lists but with updates by the F-Droid app store:

     https://f-droid.org/en/packages/com.wmstein.transektcount/
     https://f-droid.org/en/packages/com.wmstein.tourcount

Note on F-Droid:
   Getting apps from F-Droid is at least as secure as getting them from the Google Play Store.
   In contrast to the Play Store, all apps are also checked for data protection and compiled
   by F-Droid itself. If an app does not meet all of F-Droid's requirements with regard to
   undesirable features, it is noted.

   The source codes of the F-Droid apps are published and licensed with an open source license.


2. From the author's GitHub project pages (current development status)
----------------------------------------------------------------------

   Apps:
   - TransektCount app, apk file (transektcount_release_nnn.apk):
	   https://github.com/wistein/TransektCount/tree/master/apk
   - TourCount app, apk file (tourcount_release_nnn.apk):
	   https://github.com/wistein/TourCount/tree/master/apk

   Download the .apk file from the GitHub page into the Downloads directory.
   Before installing, activate on the smartphone the Android option "Unknown source" in
   "Settings -> Security". Then use the filemanager to install the app by clicking on the apk-file
   in the download folder.

   Important: Finally, deactivate the option "Unknown source"!


3. Setting up the app
---------------------

   After installation start the app once to create the necessary data directory
   Documents/TransektCount respective Documents/TourCount.
   Then get the documentation and relevant species lists from the author's GitHub pages.

   Docs:
   - TransektCount docs as well as sample species lists:
       https://github.com/wistein/TransektCount/tree/master/docs
   - TourCount docs as well as sample species lists:
	   https://github.com/wistein/TourCount/tree/master/docs

   Copy these from the download folder into the new data directory.
   Now you can restart the app and finish setting it up.


4. Notes on updates
-------------------

   Unfortunately, the two installation methods mentioned above are not mutually compatible for
   updates, since although the .apk files are generated from identical source code, they are signed
   with different keys.

   In the case of major version jumps with functional additions, structural changes may have been
   made in the internal database of an app. After such a change, the database version is incremented. 
   This is recognized by the app and the currently used DB is adapted internally.
   
   However, the currently adapted DB version cannot be used after a downgrade to a 
   previous app which uses a previous DB version.
